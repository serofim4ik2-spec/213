Установка:
1) Скачать и установить python 3.10 https://www.python.org/ftp/python/3.10.11/python-3.10.11-amd64.exe
2) Установить библиотеки, в консоль pip install -r requirements.txt
3) Если вы на Windows используйте команду: pip install ./bsddb3-6.2.9-cp310-cp310-win_amd64.whl
4) Если вы на Linux:
sudo apt-get install libdb++-dev
pip3 install bsddb3
5) для запуска в консоль python main.py

Настройка:
path_rules = путь до файла с правилами в формате хешкета. Правила нужны чтобы создавать новые пароли на основе паролей найденных в логах.
process_bads = обрабатывать ли кошельки к которым не удалось найти пароль. Если да, то софт достанет из них адрес для проверки баланса и сохранит в результаты. Полезно чтобы понять стоит ли тратить время на точечный брут по словарю.
skip_ledger = пропускать чек кошельков с леджером. Не стоит на них тратить время тк вывести крипту не получится.
skip_trezor = пропускать чек кошельков с трезором. Не стоит на них тратить время тк вывести крипту не получится.
public_addresses = глубина генерации публичных адресов.
private_keys = глубина генерации приватных ключей.
parse_mnemonics = парсить ли сид-фразы из текстовых файлах ваших логов.

Режимы:
1) Выбрать папку с логами для чека кошельков по паролям найденных в логах.
2) Точечный брут кошелька по словарю. Посмотреть информацию (hash, hash mode) для этого режима можно в Excel файле полученного из первого режима.
